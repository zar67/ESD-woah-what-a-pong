Point2D struct

A point in 2D space.

Assigns the value of another point2D to this one. Assignment constructor, that copies each field from rhs in to the lhs.

void Point2D::operator=(float[2])

Assigns a 2D float array to this point. Uses index 0,1 to copy the values from an array on to the fields used in the struct. Allows easier updates of an existing Point2D. Point2D p = {1,2}
